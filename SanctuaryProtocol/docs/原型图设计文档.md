## 设计系统 (Design System)

### 色彩方案

```
主色调：
- 背景色：#F7F5F0 (米白色，纸张质感)
- 主色/植物绿：#8B9A7C 
- 辅助色/大地棕：#D4A574 
- 强调色/暖杏色：#E8D5C4 
- 金色：#C9B896

文字色：
- 深色文字：#3D3D3D (用于浅色背景)
- 浅色文字：#F7F5F0 (用于深色背景)
- 次要文字：#6B6B6B

状态色：
- 成功：#A8C6A8 (淡绿)
- 警告：#E8D5A8 (淡黄)
- 错误：#D4A5A5 (淡红)
```

### 字体规范

```
标题：思源宋体 / Noto Serif SC / font-serif
正文：思源黑体 / Noto Sans SC / sans-serif

字号规范：
- H1: text-4xl sm:text-5xl md:text-6xl / font-serif
- H2: text-3xl sm:text-4xl md:text-5xl / font-serif  
- H3: text-xl sm:text-2xl / font-serif
- 正文: text-base / 常规
- 小字: text-sm / 常规
- 标签: text-xs / 常规
```

### 视觉风格

```
核心特征：
- 极简主义 + 东方美学
- 几何装饰元素（角落边框、旋转方块）
- 植物系配色（米白、植物绿、大地棕）
- 纸质质感背景
- 悬停时边框高亮效果

装饰元素：
- 角落装饰：┌ ┐ └ ┘ 风格的边框
- 分隔线：渐变线条 + 中心几何图形
- 悬浮效果：边框颜色变化 + 轻微阴影
```
