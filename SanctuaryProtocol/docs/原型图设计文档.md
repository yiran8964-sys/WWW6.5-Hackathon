# 原型图设计文档：Web3 植物系疗愈 OH 卡系统

## 设计系统 (Design System)

### 色彩方案

```
主色调：
- 背景色：#F7F5F0 (米白色，纸张质感)
- 主色：#8B9A7C (植物绿)
- 辅助色：#D4A574 (大地棕)
- 强调色：#E8D5C4 (暖杏色)

文字色：
- 深色文字：#3D3D3D (用于浅色背景)
- 浅色文字：#F7F5F0 (用于深色背景)

状态色：
- 成功：#A8C6A8 (淡绿)
- 警告：#E8D5A8 (淡黄)
- 错误：#D4A5A5 (淡红)
```

### 字体规范

```
标题：思源宋体 / Noto Serif SC
正文：思源黑体 / Noto Sans SC
装饰：手写体（用于特殊强调）

字号：
- H1: 32px / 粗体
- H2: 24px / 粗体
- H3: 18px / 中等
- 正文: 16px / 常规
- 小字: 14px / 常规
- 标签: 12px / 常规
```

### 圆角与阴影

```
圆角：
- 卡片: 16px
- 按钮: 24px (全圆角)
- 输入框: 8px
- 小标签: 4px

阴影：
- 卡片阴影: 0 4px 20px rgba(0,0,0,0.08)
- 悬浮阴影: 0 8px 30px rgba(0,0,0,0.12)
- 按钮阴影: 0 2px 8px rgba(139,154,124,0.3)
```

```
│                                         │
```

```
│  ┌──────────────────────────────┐      │
│  │      [1-3 图片预览]          │      │

│  ─────────── 字卡选择 ───────────       │
```

#### 交互说明

**图卡区域：**
- 图卡网格支持横向滚动或分页
- 每张图卡下方显示"已有 X 份记忆在此停留"
- 点击图卡后高亮显示，并展示在"已选图卡"区域
- "随机抽取"按钮使用伪随机算法

**字卡区域：**
- **随机字卡按钮**：点击后从30张字卡中随机选择（根据牌阵类型决定数量）
- **自定义输入按钮**：点击后弹出输入框，用户可输入自定义词汇（最多10个字）
  ```
  自定义字卡弹窗:
  ┌──────────────────────────────┐
  │  输入你的关键词              │
  │  ┌────────────────────────┐  │
  │  │  例如：释怀、突破...    │  │
  │  └────────────────────────┘  │
  │  字数: 0/10                  │
  │         [取消]  [确认添加]   │
  └──────────────────────────────┘
  ```
- 字卡可多选（根据牌阵类型限制数量，如单张牌0-1张，二元对立0-2张）
- 已选字卡显示在独立区域，可点击 × 移除
- 系统字卡和自定义字卡视觉样式区分（自定义字卡带特殊标记）

## 响应式断点

```
桌面端: > 1024px
- 图卡画廊: 5列
- 侧边栏布局

平板端: 768px - 1024px
- 图卡画廊: 3列
- 堆叠布局

移动端: < 768px
- 图卡画廊: 2列
- 底部固定操作栏
- 全屏弹窗
```

```
页面转场: fade + slide, 300ms, ease-out

按钮点击: scale(0.98), 100ms
加载动画: pulse, 1.5s, infinite
加密动画: scramble text effect, 2s
```
