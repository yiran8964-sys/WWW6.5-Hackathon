🪐 项目愿景 (Vision)
在冰冷的 Web3 时代，打造一个充满温度的、赛博梦幻风 (Cyber-Dreamy) 的数字记忆空间。它结合了“个人手工书”的私密感与“公共星河”的集体共鸣，利用区块链技术让珍贵的生命片段与成就永恒存在。

🎨 视觉风格 (Visual Identity)
核心美学： Cyberpunk x Y2K Retro-Futurism (赛博朋克结合千禧年复古未来感)。

色调： 高饱和粉 (#FF007F)、迁徙绿 (#00FF41)、深邃紫与电子黑。

质感： 磨砂玻璃 (Glassmorphism)、霓虹流光、结合传统手工书的纸张纹理。

🛠️ 核心功能 (Core Features)
1. 📖 个人讲述：私密手工书 (Personal Journal)
沉浸式编辑： 像制作手帐一样上传文字、图片与音频。

纸质感预览： 在高科技背景下保留纸张纹理和手写体美学，营造冲突的美感。

2. 🏆 成就集锦与时间轴 (Achievements & Timeline)
动态轨迹： 以流光线条串联的人生时间轴。

成就高亮： 重要瞬间可标记为“成就”，获得特殊的霓虹光晕特效。

3. 🌌 公共纪念区：霓虹星河 (Public Memorial)
粒子化展示： 每个公开的纪念片段都是星河中一颗闪烁的粒子。

互动问答： 访客可以通过“能量交换”进行提问与感悟分享，而非传统的枯燥留言。

赛博悼念： 告别黑白，用高饱和色调点亮“数字灵魂”，支持链上致敬。

4. ⏳ 赛博时间胶囊 (Time Capsule)
加密锁定： 使用区块链技术锁定的记忆，由霓虹光束缠绕，直到预定时间开启。

仪式感解锁： 开启时触发粒子爆发动效，沉浸式回顾往昔。

5. 🔗 Web3 联动 (Web3 Integration)
钱包连接： 支持 MetaMask 等钱包登录。

NFT 铸造： 将珍贵的手工书片段或纪念页面一键铸造成不可磨灭的 NFT。

互动奖励： 优质的公共区互动可获得代币积分奖励。

💻 技术栈 (Tech Stack)
前端： React.js / Vue.js + Tailwind CSS (用于快速构建高饱和度 UI)

动效： Framer Motion / Three.js (实现星河粒子效果)

Web3： Ethers.js / Wagmi (钱包连接与交互)

存储： IPFS (用于去中心化存储图片与音频，保证记忆永存)
