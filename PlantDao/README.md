PlantDAO 项目说明（社区互动强化版）

1. 项目概述

PlantDAO 是一个基于区块链的去中心化生态系统，将植物照护行为、生态价值和经济激励结合起来。系统不仅上链记录行为和生态数据，还通过 社区互动、知识共享和实物植物交换，增强人与人、人与自然的联系，实现“行为价值上链、生态价值流通”的 BioFi 模型。

2. 核心设计
2.1 双代币机制
SEED（行为价值）：通过日常照顾行为获得，用于升级植物等级、解锁功能或转换为 PLEAF
PLEAF（生态价值）：由 SEED 转换或社区贡献获得，用于购买/交换植物 NFT、参与 DAO 投票

2.2 植物类型权重与奖励公式
Effort Weight 根据植物维护难度（低、中、高）设计，低维护 0.5–0.8
奖励公式：
Reward=Base×Effort×Quality×Consistency
引入衰减机制、防刷机制，强化持续照顾

2.3 繁殖与 Offspring NFT
用户可将扦插苗、分株或种子 mint 成 Offspring NFT
NFT 包含母株来源、成长历史、照顾评分
繁殖机制受季节、环境、健康状态影响
冷却和成功率动态调整，避免滥用

2.4 植物交换与社区流通
使用 PLEAF 购买或交换植物 NFT
支持 Offspring NFT 交易，链上追溯母株来源
结合社区日记、分享、评论和点赞系统：
用户可记录照顾经验、照片、成长日志
其他用户可以点赞、评论或提出照护建议
鼓励用户共享多余种子、苗木或分株
DAO 可调节交易规则、手续费比例及特殊植物的流通限制

核心理念：通过数字化和链上激励，让园艺资源不浪费，同时促进 人与人、人与自然的真实互动

2.5 环境反馈机制
个体层：植物健康影响 SEED 奖励和繁殖成功率
局部花园层：花园整体健康对奖励加成或衰减
全局生态指数：反映系统整体健康状态
病害事件纳入生态指数（如毛茛花叶病），DAO 和社区可以投票防治

2.6 DAO 治理功能
调整 Effort Weight、季节系数、病害影响、奖励规则
决定生态阈值、交易和繁殖参数
管理社区互动规则（如点赞奖励、知识分享激励）

3. 项目特点
行为驱动生态：用户日常照顾直接影响奖励和植物健康
社区互动闭环：日记、照片、评论、点赞、分享和交易形成链上+现实的互动网络
BioFi 模型：结合生命周期、环境反馈、繁殖、交换和经济激励
可持续资源流通：解决种子、分株冗余问题，提高资源利用效率
可扩展性：支持多植物类型、Offspring NFT、季节性繁殖、植物交换和社区治理

技术栈：
Solidity + Hardhat + React + Vite + TailwindCSS + ethers.js + MetaMask
