### 1.1 产品愿景

打造一款**像素风格、以女性领袖成长叙事为核心**的 Web3 游戏。玩家通过快问快答进入特定领域（艺术、科学、法律），在领域内抽取命运卡牌并做出抉择，积累属性值，达到阈值后铸造代表品质的 SBT 徽章。集齐特定组合的 SBT 后，解锁“领袖里程碑”事件，通过正确抉择获得对应领袖 NFT 的购买资格。购买资金将自动分账至对应的公益机构，实现“游戏即捐赠”的透明闭环。

### 1.2 核心目标

- **用户体验目标**：提供沉浸式的快问快答入门，通过多阶段属性积累与徽章收集，逐步揭示领袖故事，最终获得具有情感价值的 NFT 购买资格。
- **Web3 机制目标**：将“行为证明”（SBT）与“消费权限”（白名单）深度绑定，确保只有真正认同领袖精神的玩家才能购买对应 NFT，同时实现链上透明的公益分账。
- **社会价值目标**：将游戏内消费与女性公益项目挂钩，让玩家在娱乐中为女性平权、科研、艺术等领域做出贡献。

### 1.3 用户画像

- **叙事探索者**：热爱女性成长故事，希望通过游戏了解传奇女性生平，享受逐步解锁的成就感。
- **Web3 成就猎人**：热衷于收集 SBT 徽章，关注链上资产与公益捐赠的透明记录。
- **价值观驱动者**：愿意为女性公益事业贡献资金，并希望看到资金去向的透明记录。
