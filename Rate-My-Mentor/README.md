# Rate My Mentor

A verifiable, censorship-resistant feedback layer for workplace experiences.  一套可核验、抗审查的职场体验反馈体系。

---

What is this?
这是什么？

Rate My Mentor is a protocol for **verifiable workplace feedback**.  Rate My Mentor是一个可验证的工作场所反馈协议。

Users 用户做什么？
- prove real employment experience (credential / attestation)  证明实际工作经验（证书/证明）
- submit structured reviews on company / team / mentor  提交公司/团队/导师的结构化评估

System 系统做什么？
- ensures credibility via attestation + anti-sybil  通过认证+反sybil确保可信度
- stores proof on-chain (hash, signature, timestamp)  在链上存储证明（哈希，签名，时间戳）
- stores sensitive content off-chain (encrypted)  链下存储敏感内容（加密）
  
Goal 我们的目标:
> build a trustable, non-manipulable reputation layer for workplace experience  为我们的职场体验建立一个可信的、不可操纵的 带教/所在组 声誉场
